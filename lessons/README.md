# MEX Learning System — Curriculum

## Overview

Two foundational lessons teaching ML concepts through the MEX language.
Each lesson has: explanation, examples, challenges, and tests.

## Quick Start

```bash
# List all lessons
node mex.js lessons

# Run a lesson
node mex.js lesson 1
node mex.js lesson 2

# Run a challenge
node mex.js lessons/01_challenge.mx

# Run solutions
node mex.js lessons/01_solutions.mx
```

---

## Lesson 1: What is Data?

**Concept:** Data is the fuel of ML. Every dataset has features (inputs) and labels (outputs).

**Key Terms:**
- Feature — input variable (X)
- Label — output variable (y)
- Dataset — collection of examples
- Example — one row of data

**Learning Goals:**
- Understand what features and labels are
- Load data into MEX
- Visualize data with show
- Identify patterns by looking at data

---

## Lesson 2: Finding Patterns

**Concept:** Linear regression finds the line that best fits your data.

**Key Terms:**
- Linear regression — finding a line through data
- Line of best fit — y = mx + b
- Slope (m) — how steep the line is
- Intercept (b) — where line crosses y-axis
- Training — teaching the model
- Prediction — using the model

**Learning Goals:**
- Understand how models learn
- Train a model on data
- Make predictions
- Evaluate model accuracy
